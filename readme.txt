Описание:

Данный код представляет собой программу для резервного копирования фотографий из выбранного альбома ВКонтакте. Фотографии сохраняются на Google Drive и Яндекс Диск.

Зависимости:

Для работы программы требуется установка следующих зависимостей:

requests - для отправки HTTP-запросов.
json - для работы с JSON-данными.
os - для работы с файловой системой.
tqdm - для отображения прогресса выполнения операций.
pydrive - для работы с Google Drive.
Вы можете установить зависимости, выполнив следующую команду:


pip install requests pydrive tqdm



Подготовка:

Перед запуском программы, вам потребуется выполнить несколько предварительных шагов:

Создайте проект на Google Cloud Platform и включите для него Google Drive API. Получите файл client_secrets.json, содержащий ваши учетные данные для доступа к Google Drive.

Получите токен Яндекс Диска. Это можно сделать на странице разработчика Яндекс.Диска.

Использование:

Запустите программу.

Введите свой ID ВКонтакте.

Введите свой токен Яндекс Диска.

Выберите альбом для резервного копирования фотографий.

Укажите количество фотографий для резервного копирования или введите "нет", чтобы скопировать все фотографии из выбранного альбома.

Программа начнет резервное копирование фотографий на Google Drive и Яндекс Диск. Прогресс будет отображаться.

По завершении операции, информация о фотографиях будет сохранена в JSON-файле.

Важно:

Файл client_secrets.json при скачивании имеет более расширенное имя, переименуйте именно в client_secrets.json.

Обязательно пренесите его в ту же директроию, что и основной скрипт.

Убедитесь, что у вас достаточно свободного места на Google Drive и Яндекс Диске для сохранения фотографий.

Все фотографии будут загружены в папку с названием, соответствующим выбранному альбому, на Google Drive и Яндекс.Диске.

В случае ошибок или проблем, убедитесь, что введенные вами данные (ID ВКонтакте, токен Яндекс Диска и путь к файлу client_secrets.json) верны.
